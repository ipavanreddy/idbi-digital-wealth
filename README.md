# Wealth Avatar — Track 1 (Digital Wealth Management)

An avatar-based AI wealth advisor that plugs into the bank's mobile app,
ingesting a customer's full transaction stream to become a proactive financial
coach. It categorizes spend, tracks goals, and fires event-driven nudges — with
a hybrid escalation that hands complex/regulated (SEBI) cases to a human RM as a
qualified lead, keeping the app both an advisor and a lead generator.

## Demo storyline (summary)

Synthetic customer "Ravi" with 6 months of generated transactions → salary
auto-detected → dashboard shows categorized spend → he "buys an iPhone"
(injected transaction) → avatar nudges about the dented emergency fund → he asks
about his Europe-trip goal → avatar shows the recomputed plan → he asks "which
stocks should I buy?" → avatar explains suitability and offers an RM callback →
lead is created and visible in a small RM console view.
